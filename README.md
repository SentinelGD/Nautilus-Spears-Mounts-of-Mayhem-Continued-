<summary>EN</summary>

This is a mod that transfers the elements of the Mounts of Mayhem drop to previous versions of Minecraft. In it you will find a variety of spears, nautilus, nautilus zombies, nautilus armor and many other interesting objects and creatures. This mod will add new elements to the game that will make it more intense and diverse.
